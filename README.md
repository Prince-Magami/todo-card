# Todo Task Card

This project is a simple and interactive Todo Task Card built using HTML, CSS, and JavaScript.  
It focuses on clean design, proper structure, and dynamic time updates.

## Features

- Semantic HTML structure
- All required `data-testid` attributes included
- Displays task title, description, priority, due date, and tags
- Dynamic time remaining (updates automatically)
- Checkbox to mark task as completed
- Simple modern UI with basic animations

## Technologies Used

- HTML
- CSS
- JavaScript 

## How to Run locally

1. Download or clone the project  (files)
2. Make sure they are arranged in same order as they are on Github
3. Open `index.html` in your browser
   
## Notes

- Time updates every 30 seconds
- Due date can be changed in `script.js`
